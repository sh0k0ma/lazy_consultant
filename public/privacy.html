Privacy Policy (English)

Scope. This policy applies to this public GitHub repository (including its code, documentation, issues, pull requests, wiki, discussions, releases, and—if enabled—GitHub Pages for this repository).

  • No data collection. We do not collect, store, process, sell, or share any personal data about users of this repository. We do not use cookies, analytics, tracking pixels, advertising IDs, telemetry, or fingerprinting.
  • Public content you choose to publish. Anything you voluntarily post (e.g., issues, pull requests, comments, discussions, commits) is public by design on GitHub. Do not include information you do not wish to make public.
  • Hosting provider (GitHub). This repository is hosted on GitHub. GitHub may process data under its own terms and privacy statement. See GitHub's Privacy Statement: https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement.
  • GitHub Pages (if used). If this repository publishes a static site via GitHub Pages, we do not embed third-party analytics or tracking. GitHub or your browser/network may still log requests as part of normal web operations.
  • Children. This repository is not directed to children. We collect no data regardless of user age.
  • Changes. We may update this policy; changes will be reflected by commits to this repository and the "Last updated" date above.
  • Contact. To reach the maintainers, open a GitHub issue in this repository or visit https://shokoma.com/contact.

プライバシーポリシー（日本語）

対象範囲：本ポリシーは、本GitHub公開リポジトリ（コード、ドキュメント、Issues、Pull Requests、Wiki、Discussions、Releases、および必要に応じて本リポジトリのGitHub Pages）に適用されます。

  • ユーザーデータは一切収集しません。当方は、本リポジトリの利用者に関する個人データを収集・保存・処理・販売・共有しません。Cookie、アクセス解析、トラッキングピクセル、広告ID、テレメトリ、フィンガープリンティング等も使用しません。
  • 投稿は公開情報になります。Issues・Pull Requests・コメント・Discussions・コミット等、利用者が任意に投稿した情報はGitHub上で公開されます。公開を望まない情報は投稿しないでください。
  • ホスティング（GitHub）について。本リポジトリはGitHub上でホストされています。GitHubによるデータ処理はGitHub独自の利用規約およびプライバシーステートメントに従います。GitHubのプライバシー：https://docs.github.com/ja/site-policy/privacy-policies/github-privacy-statement
  • GitHub Pages（利用時）。本リポジトリがGitHub Pagesで静的サイトを公開する場合でも、第三者の解析/トラッキングを埋め込んでいません。なお、通常のWeb動作としてGitHubまたはブラウザ/ネットワークによるリクエストの記録が行われる場合があります。
  • 児童向けではありません。本リポジトリは児童を対象としていません。年齢にかかわらずデータ収集は行いません。
  • 変更。本ポリシーは予告なく更新される場合があります。変更はリポジトリのコミットおよび上記「最終更新日」に反映します。
  • 連絡先。メンテナに連絡する場合は、本リポジトリにIssueを作成するか、https://shokoma.com/contactまでお問い合わせください。

Политика конфиденциальности (Русский)

  • Мы не собираем данные. Мы не собираем, не храним, не обрабатываем, не продаём и не передаём персональные данные пользователей этого репозитория. Мы не используем cookies, аналитику, трекеры, рекламные пиксели, телеметрию или «фингерпринтинг».
  • Публичный контент по вашему выбору. Всё, что вы добровольно публикуете (issues, pull requests, комментарии, обсуждения, коммиты), по своей природе является публичным на GitHub. Не публикуйте сведения, которые вы не хотите раскрывать.
  • Провайдер хостинга (GitHub). Репозиторий размещён на GitHub. GitHub может обрабатывать данные на основании собственных условий и политики конфиденциальности: GitHub Privacy Statement.
  • GitHub Pages (при использовании). Если репозиторий публикует статический сайт через GitHub Pages, мы не внедряем стороннюю аналитику или трекинг. Тем не менее GitHub или ваш браузер/сеть могут вести журналы запросов в рамках обычной работы веба.
  • Дети. Репозиторий не предназначен для детей. Мы не собираем данные вне зависимости от возраста пользователя.
  • Изменения. Мы можем обновлять эту политику; изменения отражаются в коммитах и в дате «Последнее обновление» выше.
  • Контакты. Создайте issue в этом репозитории или посетите https://shokoma.com/contact.

隐私政策（中文）

适用范围：本政策适用于本 GitHub 公开仓库（包括代码、文档、Issues、Pull Requests、Wiki、Discussions、Releases，以及（如启用）本仓库的 GitHub Pages）。

  • 不收集任何用户数据。我们不收集、存储、处理、出售或分享任何与本仓库使用者有关的个人数据。我们不使用 Cookie、分析工具、跟踪像素、广告 ID、遥测或浏览器指纹等技术。
  • 您自愿发布的公开内容。您在 Issues、Pull Requests、评论、Discussions 或提交（commit）中自愿发布的信息，将在 GitHub 上公开展示。请勿发布不希望公开的信息。
  • 托管服务（GitHub）。本仓库托管在 GitHub。GitHub 可能依据其自身条款与隐私声明处理数据。请参见：GitHub 隐私声明。
  • GitHub Pages（如使用）。若本仓库通过 GitHub Pages 发布静态网站，我们不嵌入任何第三方分析或跟踪脚本。但作为正常的网络运行，GitHub 或您的浏览器/网络可能会记录请求日志。
  • 未成年人。本仓库并非面向未成年人；无论年龄如何，我们均不收集数据。
  • 变更。我们可能更新本政策；更改将通过仓库提交记录与上述"最后更新"日期予以体现。
  • 联系。请在本仓库创建 Issue，或访问 https://shokoma.com/contact。
