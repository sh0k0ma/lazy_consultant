# Lazy Consultant

**コンサルティングワークを効率化するプロジェクト管理ツール**

---

## 📋 概要

Lazy Consultantは、コンサルタント業務に特化したプロジェクト・タスク管理システムです。あらゆる業務に適用できるユニバーサルフレームワークを提供し、プランニング・エグゼキューション・クロージングの3フェーズで体系的に業務を進められます。

### 主な機能
- **プロジェクト・ワーク・タスク管理**: 階層的な業務管理
- **ユニバーサルフレームワーク**: 3フェーズ構成の汎用的なタスク管理
- **ナレッジベース**: 再利用可能な知見の蓄積
- **ジャーナル機能**: 会議メモや進捗の記録
- **データエクスポート**: JSON形式での一括出力

## 🎯 ユニバーサルフレームワーク

すべてのタスクに適用できる3フェーズ構成のフレームワークです。

### 1. プランニング（Planning）
**目的設定・課題分析・計画立案のフェーズ**

- 目的・背景・成功基準の明確化
- 課題の特定と構造化
- アプローチ・体制の設計
- スケジュールとマイルストーンの策定
- ステークホルダーマップの作成
- リスク分析と対策

計22のチェック項目で、業務の土台をしっかり固めます。

### 2. エグゼキューション（Execution）
**実行・検証・調整のフェーズ**

- デイリーオペレーション
- 進捗の可視化と報告
- 品質管理とレビュー
- ステークホルダー対応
- 課題管理と是正措置
- 変更管理とリスク対応

計17のチェック項目で、実行品質を維持します。

### 3. クロージング（Closing）
**完了・引き継ぎ・振り返りのフェーズ**

- 成果物の検収
- ドキュメント整備
- 運用への引き継ぎ
- 振り返りとナレッジ化
- ステークホルダーへの報告
- プロジェクト完了手続き

計12のチェック項目で、確実なクロージングを実現します。

---

### フレームワークの特徴

- **汎用性**: 施策立案、課題解決、リサーチなど、あらゆる業務に適用可能
- **網羅性**: 51のチェック項目でコンサルティング業務の要点を網羅
- **柔軟性**: 必要な項目を選んで使用できるチェックリスト方式
- **進捗管理**: フェーズごとに完了状態を管理し、進捗を可視化

## 🛠️ 技術スタック

- **バックエンド**: Node.js + Express
- **フロントエンド**: Vanilla JavaScript (ES6 Modules)
- **データ保存**: JSON ファイル
- **スタイル**: カスタムCSS（CSS Variables使用）

外部フレームワークに依存しないシンプルな構成で、高速かつ拡張性の高い設計です。
